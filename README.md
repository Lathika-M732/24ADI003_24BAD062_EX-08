SCENARIO 1 – ASSOCIATION RULE MINING USING APRIORI ALGORITHM
DATASET LINK:https://www.kaggle.com/datasets/heeraldedhia/groceries-dataset
      This project focuses on discovering meaningful patterns in transactional data using the Apriori algorithm for Association Rule Mining. The goal is to identify frequently occurring itemsets and generate strong association rules that reveal relationships between items purchased together.
By analyzing customer purchase behavior, the model uncovers patterns such as “customers who buy item A are also likely to buy item B.” These insights are widely used in market basket analysis to improve product placement, cross-selling strategies, and recommendation systems in retail environments.

SCENARIO 2 – DIMENSIONALITY REDUCTION USING PCA
DATASET LINK:https://www.kaggle.com/datasets/heeraldedhia/groceries-dataset
      This project focuses on reducing the complexity of high-dimensional data using Principal Component Analysis (PCA). The goal is to transform the original features into a smaller set of uncorrelated variables called principal components, while retaining most of the important information (variance) present in the data.
By applying PCA, the dataset becomes easier to visualize, faster to process, and less prone to overfitting. This technique helps in identifying the most significant patterns in the data and is widely used as a preprocessing step in machine learning tasks.
